Code Analysis

Efficiency: Unlike a standard loop that checks every single bit (e.g., 32 times for an int), this loop only runs once for every '1' present in the number.

If you have a 64-bit integer like 1000000000000000 (which has only one bit set), a normal loop would run 60+ times. This algorithm finishes in exactly one step.
